为什么要爱：
 - 我们年轻时候告诉我们说，爱情是乐园，是个糖水铺，但经典文学一定告诉你这句话，爱情是我们人生的一个新的阶段和新的命题。我们要在爱情里面，他在我们人生当中的意义是什么呢？他是教我们成长的。


单身是一种状态，恋爱也是一种状态。哪怕拥有了伴侣，我们也是在这两种状态中不断切换的。步入关系之前，要学会一个人生活，拥有一个自洽的内心世界。
- 许多人将爱情视为填补内心空洞的灵药，却不知带着匮乏感进入关系的人，往往会陷入“寄生式恋爱”————一方拼命索取，另一方逐渐窒息
- 期望会把接受和让人自由等充满爱意的感觉挡在门外
  - 依恋型人格包括焦虑型和回避型：
    - 情感需求过度：需要对方随时回应消息，不回消息就焦虑不安
    - 过度寻求确认：不断询问"你爱我吗？""你是不是不在乎我了？"
    - 害怕分离：对方稍微晚回消息或工作忙，就感觉被抛弃
    - 丧失自我：为取悦对方放弃自己的兴趣和社交圈
    - 情绪波动大：心情随对方态度起伏，容易陷入抑郁或焦虑  
    - 如果你真的爱我，你就会……
      - 期待对方秒回消息：不回消息就认定对方不爱自己
      - 期待对方"读懂自己"：不需要表达，对方就能知道自己的需求
      - 期待对方永远把自己放在第一位：要求对方放弃自己的朋友、工作、兴趣

爱情关系中，两个人都应该是独立的。如果绑得太紧，爱情就容易变成一种消耗品。（你想在这棵树上吊死，你要考虑一下这棵树的感受啊）


认识自己：
- 人生有两大课题：认识自己，认识他人。很多人跳过了第一个课题而直接步入第二个课题。我们需要了解自己的性格、需求、情绪和情感。
- 我们缺乏"单身教育"和自我认知的引导。自我认知教育应该从幼儿园就开始，先认识自己，再认识这个世界。
- 在当今社会，成年人最好的自我认知方法往往是经历痛苦，在遗憾中重新思考自己真正需要什么
- eg.很多人忽略了独自生活的成长过程————搬家租房、自己做饭、处理和小姐孤独等，这些都是重要的自我成长环节



- 所有关系都需要付出和承担，包括承受对方的情绪。
  - eg.暧昧关系，虽然它只能够弥补很小一部分的孤独感，同时它的但付出和责任都很低
- 好的恋爱不是消耗性的，虽然会消耗时间、金钱等物理性东西，但精神上应该是更加振奋的，在健康的关系中，世界会显得更加美好，生活充满能量。恋爱以后，我觉得我连领导的脸都顺溜了很多。这个是什么？就是爱情的能量的赋予。
- 当代很多年轻人没有投身到一段关系中的精力、能力或决心，但内心的孤独又在隐隐作痛



"恋爱脑"：
- 将爱情视为最高价值观念是"恋爱脑"的表现。这种情况下，人们爱的往往不是真实的人，而是自己的幻想。跟一个不真实的人谈恋爱，很难有好的结果。当你与心中的"神"谈恋爱时，自己的地位也高不到哪里去，往往会成为讨好型人格，所以我们经常能够发现，恋爱脑和舔狗往往相互吸引。
- 另一个普遍特点是：当一段关系已经对自己造成较大伤害，却为了维持关系而无底线地原谅和宽容对方。
- 如何在恋爱中守住自己的底线，是每个人都必须面对的课题，但是这些都不是先通过观念接受来的，一定是通过我要在这个真实的关系相处当中得来的。
    - 讨好不能换来真爱；
    - 恋爱前三个月的滤镜不是自然状态，更不是永恒状态；
    - 不能把所有个人价值都寄托在一段关系上。

很多人被"花束般的恋爱"所迷惑，这种关系建立在标签和消费上，很精致却容易枯萎。当关系进入实际生活阶段——家务分配、工作压力、经济负担等问题出现时，就容易出现问题。我们不应该把文艺作品中建构的情感观念当作真实的情感标准，而应该在真实关系中体验、打碎和重建，而且甚至于这个过程都是需要反复和持久的。


我记得前面曾经在某一次对谈当中，他们有人问我说，当时你觉得这一代年轻人跟前面几代什么地方不一样，我觉得他们最不一样的是，他们有一个特殊的使命，非常明显，他们要重建这个世界的认知。这就是为什么以前在前几代人觉得很正常的东西，这一代人不接受了，因为他们要重建认知。但是他没有话语权，所以他们经常会用一些很奇怪的方式来表达自己的观点。你会发现，网络语特别多，年轻人造的词也特别多。你比如我举一个简单的例子，我昨天还在百度一个词叫老登。不是，你看我们那一代人一直生活在那个前辈人的那些爹味儿的人的高压之下，我们那代人就从来没有发明过爹味儿和老登这个词。因为我们会觉得很正常，他们剥夺你的话语权，然后来控制你，他们什么都对，你会觉得很正常。这一代年轻人为什么会发明出来了呢？因为他觉得这个不正常。我觉得这就是这一代年轻人比较伟大的地方。

经典文学对人的塑造力量很大。《允许爱情消失》这本书选取的都是经典文学，他们谈论爱情的方式往往是"祛魅"的，即使讲述热烈的爱情故事，作者本身对爱情保持清醒认知。经典之所以成为经典，是因为它讲述的是恒久的人性规律、命运规律和生活规律。好的爱情能让人成长，即使是糟糕的爱情，也能通过碰壁获得成长。阅读经典让我们通过不同人物的人生经历，积累对人性和关系的理解。跟着一个人物，你又活一回，你读了一百本小说，就等于活了100百回，所以你怎么可能还不了解这些人生的。你读的小说太多了，你看了太多人生的现象了，太多命运的现象了，我们来把握自己，来驾驭自己的情感，驾驭自己的情绪，就不会那么难了。

爱情应该是自然或偶然的，而不是数据匹配的结果。人生主题应该是不断探索自己、建设自己、享受这个过程。在自我建设的路上，如果遇到了一个人，可以共同走一段路，无论长短，哪怕是一个月，一天，都是美好的，因为它会滋养我们。爱情不应是必需品，而是生活中的珍稀品——得到了珍惜，得不到也能保持完整。

社交软件可以作为拓展社交圈的工具，但不应执着于它。关系从来不是规划的产物，因为它包含太多偶然性和随机性。关系是一种体验过程，需要活在当下。使用社交软件时，可以抱着见证人类多样性的开放心态。关系的深度行业契合度都不是计划出来的，它是一个必须的过程中有各种各样的偶然和随机。

其实关键问题不是实际问题，而是你的观念问题。

"磨合"这个词其实很痛。亲密关系中，两个人在一起需要的是充分沟通的过程，甚至吵架都比所谓的"磨合"好，因为吵架也是沟通。"吵商"很重要，沟通和吵架是在交换立场和角度。人与人之间最大的障碍不是差异本身，而是不了解对方的角度和立场。我们年轻人是在传统家族里面长大的，一般在两个人相处遇到矛盾的时候就会生闷气。生闷气非常可怕。爱我的爱人，唯一的方式就是我要跟他沟通。我永远持有一份跟你沟通的耐心和勇气，这就是爱。如果这样的一份沟通下来，发现我们依然不合适，那就别将就了，立即分手。
    当然我们来判断对方是不是一个值得交往，值得走下去的人。你也是这是个核心点，你看他愿不愿意跟你沟通，他愿不愿意了解你的说话背后的意思，或者说你们俩愿不愿意一起不断提升我们互相沟通的技巧，或者我们的坦然度。

爱一个人最重要的表现是愿意与他沟通，愿意了解他的角度、立场和局限。被爱的感觉不是收到多少礼物，而是感受到对方聆听和理解自己的能力。
    我告诉大家一个概念，你单身要是比起婚姻里面的那种寂寞感，那好的多了，你明明两个人就有婚姻，但是你的观念想法没有人聆听，好像那个人存在，又不存在，那种感觉才是致命的。所以这个也是我们判断爱情好和不好的一个小小的诀窍。

不是必须和父母搞好关系，而是应该先与自己和解，因为与自我的关系就是与世界的关系，是最核心的。

占有欲和占有是两回事。占有本质上是不在乎对方，而是需要通过控制来体现"爱"。真正的爱体现在好奇心上——对伴侣的一切都感兴趣，愿意倾听他的"废话"。

在亲密关系中，要特别注意避免说那些明知会深深伤害对方的话，因为这些话一旦出口，往往无法挽回。那个关键节点你能不能忍住，所有的亲密关系相处的过程中，包括跟父母相处的过程中，你知道有一句话你能毁掉他。能够让他暴跳如雷，立马模仿，摔门而出。

如果为了恋爱而减少与朋友的交往，本质上是被缩限在关系中，对方的控制会越来越强，自我价值会过度依赖这段关系。谈恋爱后，生活状态不应有太大变化，因为对方是同行者，而不是价值的剥夺者。分手时的痛苦往往源于将自我价值投射到对方身上————不是因为对方有多少优点，而是因为自己配得感低，将自己的优点投射到了对方身上。

传统爱情观念中有许多需要纠正的错误。比如，选择伴侣时应关注他多么了解你、能给你多少能量、是否听懂你的需求，而不是比较对方有多少车有多少房，多高，会不会游泳，有多少外在优点。
我们传统爱情观念当中太主张一个人对另外一个人的牺牲了。如果我爱他，就是我们不懂得这么爱一个人，传统的时候我爱一个人，我就会变成他妈，我要爱他，以后会变成他爸。然后把对他照顾成一个残废，然后啥都给他弄好，这叫我爱他。爱情不是牺牲和照顾，真正的爱情包含两点：第一，"我爱你"意味着"我是最愿意和你沟通、最能听懂你的人"；第二，如果我爱你，我会努力成长为一个能与你共同成长的人，而不是互相消耗的人。最正确的爱情观正如《致橡树》所述：你是一棵树，我也要长成一棵树，和你并肩站立，各自向着天空生长，让枝叶在空中交叠。

每一代人的家庭和婚姻观念都与当时的经济背景紧密相关。在物质匮乏年代，婚姻更多是解决生存问题的工具。这就是为什么我们的老一代会说这个人适合结婚，他这个地方指的结婚的概念是在生存工具的概念上，条件好，但是这个很有害。因为这个婚姻观念的确它可以解决我们的生存问题，但是它风险很大，他有可能带给你一个无趣的寂寞的。假如你对方不够爱对方，也不够爱你，你们之间的契合点比较少，你不要狂妄的认为你可以熬过那些岁月，到婚姻的后半段，婚姻会变成为了解决生存而缔结的婚姻，有可能变成凌迟之行。今天，我们需要明确自己的期待：如果婚姻只为解决生存问题，工具式婚姻可以接受；但如果期待情感满足，就不应进入缺乏精神交流的婚姻。许多人以为的"现实主义"其实并不现实——没有精神交流的婚姻不仅带来情感凌迟，在现实层面（如离婚时的财产分割）也可能处于极度不利地位

这个人条件有多好，本质上其实跟你是无关的，因为他的条件好可以给你，也可以给另外一个人，如果幸运，你可以分有一部分，你住在我们家房子里面，一旦他变心，可能这房子你就没有办法去占有。

人活的时间很短，你到一定岁数，他怎么这么快，我就这个岁数了。然后再往后看看，也许我身边很多人就不在了，其实值得过的日子无比简单：让自己感到愉悦。不添堵就是最好的婚姻。如果一个人不是坏人，却天天让你糟心，为什么要将短暂的一生与他绑定？人生要有全局观，看到生命的终点。爱自己的标志是不再需要讨论"什么是爱自己"——真正快乐的人不会整天思考如何快乐。

当一个人自洽且内心丰富时，真正允许开始的爱情是世界上最好的爱情。因为这不是为了满足孤独或解决生存问题，而是出于情感上的真实碰撞和高端满足。如果在考虑是否要因物质条件不足而放弃恋爱，或许应该反思：游戏、电影、工作和自我提升同样能带来快乐。当恋爱需要以大量物质付出为前提时，可能应该先专注于自我发展。你现在有一个人站在你面前，让你考虑说我要允许我自己跟他开始吗？当你这样想的时候，我觉得可能还不够打动你。因为如果一个真的跟你契合的人，可能你不需要考虑这个，你不会问我要不要允许他，因为你一定会开始。

我们需要先改变"单身不合法"的观念。当不接受单身的合法性时，人们会不惜一切代价寻找伴侣，这源于"人不可以单身"的社会压力。其实，一个人的生活也可以很完整、值得尊敬且富有创造性。有些人说要结婚是为了完成父母期待或社会要求，这样的人真的适合进入亲密关系吗？感情这件事，一定要抢夺定义权——如果不定义自己想要的感情，社会就会替你定义，你的人生就会被别人掌控。

一个父母你有什么权利去要求他说你赶紧恋爱，赶紧结婚，结婚又不是说你给他奖金。父母催婚虽出于关心，但婚姻中的苦楚他们不会替你承担。所以还是要建立主体性，自己做主人生非常重要。

不是必须和父母搞好关系，而是和自己和解开始，毕竟与自我的关系就是与世界的关系，是最核心的。

最后有一点就是你再恋爱，一定要允许爱情消失了。就是你不能够把你的所有个人价值保护到这儿，去确认什么东西是唯一性的。没有他你就活不了的。

◆ 也就是说，在闲聊中，避免做结论性的发

◆ 言，是让闲聊愉快地进行下去、进而加深

◆ 因此，在闲聊时，请不要统整话题，不要做出任何抽象的、普遍观点的结论；要避免“总结”，让话题自由地向各个方向延伸。

◆ 称赞对方的领带花纹好、衣服款式好、口红颜色好
